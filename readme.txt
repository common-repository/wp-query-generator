Generate queries from wp-admin

== Description ==
this simple plugin will allow you to generate WordPress queries right from you wp-admin

== Installation ==
1. Upload `azi_query` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \\\'Plugins\\\' menu in WordPress
